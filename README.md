# 프로잭트 소개

호남을 잇;EAT다. 호잇은 펜데믹이 정리되는 분위기 속 최근 국내여행객들이 늘어나 관광 산업이 다시 활성화 되는 추세입니다. 그 중 호남지역으로 혼자 여행을 가는 사람들을 타겟으로 동행자와 여행지를 매칭해주는 서비스입니다.
